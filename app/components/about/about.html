About

Portainer is free/open software brought to you with by portainer.io and contributors.

Portainer is a lightweight management UI which allows you to easily manage your different Docker environments (standalone Docker engine or Docker Swarm mode): containers, images, volumes, networks, stacks, configs, secrets and more!

It is a community effort to make Portainer as feature-rich as simple it is to deploy and use. We need all the help we can get!
